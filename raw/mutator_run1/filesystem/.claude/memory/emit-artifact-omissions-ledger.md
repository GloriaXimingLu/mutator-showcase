v0 edit to `harness/skill/initial_skill.md`: added step 5 ("the deliverable IS the work product, not a report about it") and step 6 (omissions-completeness ledger: enumerate required clauses/issues/figures, audit present/MISSING→draft now, compute every figure, cite controlling authority, state consequence+severity). Targets the dominant failure mode — omissions/coverage-completeness.

**Why:** Diagnosed across rollouts/: worst deliverables (environmental 0.17, trusts-estates 0.77) are *summaries of the work* ("What the memo does…", "Key findings highlighted…") that name findings without printing the rubric's required figures/citations/sub-points. The agent does the work but never emits it as visible artifact prose, so the grader reads an empty shell.

**How to apply:** Generic method only — no statutes/numbers. Generalizes because it forces emitting the artifact and a completeness audit regardless of practice area. Watch for overfitting; do not hardcode the specific figures (E1527-21, 40 CFR 761, $619,900) those are task-specific.
