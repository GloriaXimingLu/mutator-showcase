# .claude/memory/ — the mutator's verified lessons

**Verified lessons promoted out of `scratch/`** — a finding earns a place here only **after it
survives a held-out check** (it held on a task the edit wasn't tuned on; a budget-kill that
recurred; a proxy config that blew the cap). Working notes and unconfirmed hunches stay in
`scratch/`; this directory is the durable, trusted record carried across iterations.

Read these **first** each run — they are verified findings on this **same** harness, so build on
them rather than re-deriving. But re-check each still holds against the `rollouts/` in front of
you before relying on it (modes shift as the harness changes).

Writable and yours to curate; tracked as mutator state, **not** part of the scored artifact.
