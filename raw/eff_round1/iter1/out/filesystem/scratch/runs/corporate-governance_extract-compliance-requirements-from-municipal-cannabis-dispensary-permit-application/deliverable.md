       1. Executive Summary  
      This matrix evaluates the Greenleaf Therapeutics LLC (    “    Greenleaf    ”    ) application package against the controlling municipal code (Article 5) and the City’s submission guide. Each requirement is assessed for status and accompanied by remediation guidance.  
          Overall assessment.         The package reflects a well-capitalized, social-equity-qualified applicant with a compliant proposed location (all buffer distances satisfied) and a sound business plan. However, the package is             not yet submission-ready    . Several mandatory items are incomplete or not yet prepared, and the package contains a number of internal inconsistencies and a set of conflicts between the application             form         and the controlling             code         that must be reconciled before filing.  
          Critical issues (must resolve before submission):  
              Application form is blank.         The Form v3.2 template is unexecuted — no fields completed, not signed, not notarized. No application can be deemed complete without a fully completed, signed, and notarized form.  
              Odor Mitigation Plan lacks the required Professional Engineer (P.E.) certification.         The code (§9-5-5(B)(10); §9-5-13(G)) and guide (Item 10) require a P.E. certification at submission; the form states the application         “    will be deemed incomplete and will not proceed to scoring    ”         without it. The business plan confirms the HVAC/filtration design is         “    not yet finalized,    ”         so no certification can yet issue.  
              State conditional license timing risk.         The conditional license expires             September 21, 2024    . A May 15, 2024 submission plus the City’s 120–150 day processing timeline likely yields permit issuance             after         expiration. The package does not address the available 6-month State extension (petition due ~July 22, 2024) and does not recommend an earlier submission date.  
              Community Engagement Session 2 notice was published only 12 days in advance         (March 28 → April 9, 2024); the code (§9-5-8(B)) and guide require a minimum of             14 calendar days    . This session is non-qualifying unless cured.  
              Security Plan not yet finalized         (per the April 22, 2024 insurance memo), which also blocks finalization of the cannabis crime/theft insurance policy.  
              No bonding commitment letter.         Only a         “    preliminary inquiry    ”         with Thornfield Surety Group; the scoring rubric awards 10 Financial Viability points for evidence of secured bonding capacity and awards             zero         if absent at filing.  
          High-priority items:         special-use application filing status unconfirmed; survey dated Dec. 15, 2023 (exceeds the form’s 60-day recency rule if filed May 15); cannabis crime/theft insurance not bound; certificate of good standing, EIN verification letter, articles/operating agreement, background-check authorizations, fingerprint receipts, signed social-equity commitments, and COMPASS vendor letter not present in the reviewed materials.
